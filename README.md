# 电商美妆区域销售分析
## 项目简介
本项目基于电商平台销售订单数据（3.1万条记录），使用 MySQL进行数据清洗和多维度分析，使用 Excel完成数据可视化，输出可落地的业务建议。
## 技术工具
| 工具 | 用途 |
|------|------|
| MySQL | 数据清洗、多表关联、分组聚合、窗口函数 |
| Excel | 数据透视表、柱状图、饼图、条形图 |

## 数据说明

- 数据量：3.1万条记录
- 核心字段：订单编码、所在区域、所在省份、所在地市、商品编号、订购数量、金额
- 关联表：商品信息表（商品编号 → 商品大类、商品小类）

## 分析内容

### 1. 全国层面分析

各省份护肤品/彩妆销量排行，识别核心市场。

**核心发现**：江苏、广东、浙江为TOP3省份，合计贡献约35%总销量。

### 2. 窗口函数应用（ROW_NUMBER）

使用窗口函数对各省份内所有城市按护肤品销量进行降序排名，识别每个省份的“销冠城市”。

**核心SQL代码：**
```sql
SELECT 
    s.所在省份,
    s.所在地市,
    SUM(s.订购数量) AS 总销量,
    ROW_NUMBER() OVER (PARTITION BY s.所在省份 ORDER BY SUM(s.订购数量) DESC) AS 省内排名
FROM sales_orders s
JOIN products p ON s.商品编号 = p.商品编号
WHERE p.商品大类 = '护肤品'
GROUP BY s.所在省份, s.所在地市
ORDER BY s.所在省份, 省内排名;
